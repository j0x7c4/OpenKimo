## [0.1.0] - 2026-04-27

### Added
- Initial release of OpenKimo.
- Docker Compose one-command deployment (`docker compose up`).
- Web-based session management UI.
- Multi-LLM support: Kimi, OpenAI, and Anthropic.
- Mandatory Docker sandbox isolation for every agent session.
- Jupyter Kernel integration for secure code execution.
- Headless Chromium browser automation.
- Resource limits enforcement: CPU, memory, disk, and PID cgroups.
- Dangerous command interception and blocking.
